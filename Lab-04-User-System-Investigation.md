# Linux Lab 4 – User & System Investigation

## Objective
Learn how to identify the current user, view user information, and review command history in Linux.

## Commands Used

```bash
cd ~
mkdir user-system-lab
cd user-system-lab
pwd
whoami
id
history
history | tail
history | grep find
history | grep grep
```

## Skills Learned

- Identifying the current logged-in user
- Viewing User ID (UID) and Group ID (GID)
- Checking user groups
- Viewing command history
- Filtering command history using `grep`

## Key Concepts

### `whoami`
Displays the username of the currently logged-in user.

### `id`
Shows the user's UID, GID, and group memberships.

### `history`
Displays previously executed commands.

### `history | tail`
Shows the last 10 commands.

### `history | grep`
Searches for specific commands in the history.

## Result

Successfully investigated user information and reviewed command history using Linux commands.
