# Linux Lab 2 - Search Files and Text

## Objective
Practice searching for files and finding text using Linux commands.

## Commands Used

```bash
mkdir search-lab
cd search-lab
touch user.txt
touch log.txt
touch notes.txt
find . -name "*.txt"
grep "Linux" notes.txt
grep "Humaima" user.txt
```

## Skills Learned
- Creating directories
- Creating files
- Searching for files with `find`
- Searching text inside files with `grep`

## Result
Successfully searched files and text using Linux commands.
