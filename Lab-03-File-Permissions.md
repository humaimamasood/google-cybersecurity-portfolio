# Linux Lab 3 - File Permissions

## Objective
Learn how to view and modify file permissions in Linux.

## Commands Used

```bash
mkdir permissions-lab
cd permissions-lab
touch secret.txt
touch public.txt
ls -l
chmod 600 secret.txt
ls -l
chmod 644 public.txt
ls -l
stat secret.txt
stat public.txt
```

## Skills Learned

- Viewing file permissions
- Changing permissions using `chmod`
- Understanding Linux file ownership
- Viewing file information with `stat`

## Result

Successfully viewed and modified file permissions using Linux commands.
